# Python Backend Learning

>  Практический репозиторий для изучения и отработки навыков backend-разработки на Python.


## О проекте
Этот репозиторий — мой личный дневник для изучения backend-разработки на python. Здесь я буду:
- Пишу и тестирую REST API
- Работаю с БД, миграциями и ORM
- Реализую аутентификацию, авторизацию и бизнес-логику
- Осваиваю Docker, CI/CD и лучшие практики продакшена
- Покрываю код тестами и документацией

> Код может быть экспериментальным. Коммиты и ветки отражают процесс обучения и рефакторинга.
